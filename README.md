# Makerspace traffic detector

## Tools

1. Arduino Uno
1. UnaShield V2S
1. Infrared transmitter IR333-A
1. Infrared module TSIP 1738
1. BJT NPN transistor TIP 120
1. Adapter with DC output. Rating: 5V 1A
1. 330 ohm resistor
1. Potentiometer
1. 4 x AA Battery Holder
1. Jumper Wire
1. Breadboard/Donut board
